## 🛠️ Troubleshooting

### The game will not open
- Make sure you extracted the zip file first.
- Check that the file is a Windows app, not a web page.
- Try right-clicking the file and choosing Run as administrator.

### The screen is blank or black
- Restart the game.
- Update your graphics driver.
- Switch between windowed and full-screen mode.

### The mouse scratch feel is not smooth
- Close other apps.
- Lower graphic settings if the game has them.
- Use a direct mouse instead of a trackpad.

### The game closes right away
- Re-download the file.
- Make sure the download finished.
- Check whether Windows blocked the app.

### Sound does not play
- Check your Windows volume.
- Open the game audio settings.
- Make sure your speakers or headset are active.
