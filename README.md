## 🚀 Quick Start (Recommended)
1. Extract the archive to any folder
2. Run `ScritchyScratchy.exe` (run as Administrator first time for smooth setup)
3. Start with basic cards → drag mouse to scratch (satisfying ASMR vibes!)
4. Earn cash → buy better cards & upgrades
5. Unlock **Auto Scratcher** → let it scratch automatically while you idle
6. Prestige when ready → multiply earnings forever
7. Chase jackpots – go all-in or play safe?

> ℹ️ Game supports offline progression – Auto Scratcher keeps working even when closed!

## ❗️ Troubleshooting
| Problem                        | Solution                                                                 |
|--------------------------------|--------------------------------------------------------------------------|
| Game doesn't launch            | Run as Administrator; check antivirus exclusions (false positive common)|
| No sound / low FPS             | Update graphics drivers; lower resolution in settings                    |
| Saves not working              | Run from non-protected folder (not Program Files); check antivirus      |
| Scratch feels laggy            | Enable V-Sync off or cap FPS; use Performance mode                      |
| Missing cards / features       | This is full v1.0 – prestige to unlock more                             |
| Antivirus flags exe            | Add to exclusions – common for indie .exe files                         |
